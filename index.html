先决条件

• 三台CentOS 7+ 服务器

• 网络互通（同一网段）

• 管理员(root)权限

• JDK安装包 (jdk-8u351-linux-x64.tar.gz)

主机名与网络配置

1. 设置主机名

在三台服务器上分别设置主机名为node1、node2、node3

sudo hostnamectl set-hostname node1  # 在node1上执行
sudo hostnamectl set-hostname node2  # 在node2上执行
sudo hostnamectl set-hostname node3  # 在node3上执行

2. 配置静态IP

编辑网络配置文件（示例为ens33网卡）

vim /etc/sysconfig/network-scripts/ifcfg-ens33

修改为以下内容：

TYPE="Ethernet"
PROXY_METHOD="none"
BROWSER_ONLY="no"
BOOTPROTO="static"
DEFROUTE="yes"
IPV4_FAILURE_FATAL="no"
IPV6INIT="yes"
IPV6_AUTOCONF="yes"
IPV6_DEFROUTE="yes"
IPV6_FAILURE_FATAL="no"
IPV6_ADDR_GEN_MODE="stable-privacy"
NAME="ens33"
UUID="bdbe97ce-0a07-40fb-a611-02dceb78c04d"
DEVICE="ens33"
ONBOOT="yes"
IPADDR="192.168.88.101"  # node2用102, node3用103
NETMASK="255.255.255.0"
GATEWAY="192.168.88.2"
DNS1="192.168.88.2"

应用网络配置更改

systemctl restart network
ifconfig  # 验证IP配置

注意： 请根据实际网络环境调整IP地址、子网掩码和网关

主机名映射

1. Windows主机配置

编辑Windows系统中的hosts文件

路径：C:\Windows\System32\drivers\etc\hosts

192.168.88.101 node1
192.168.88.102 node2
192.168.88.103 node3

2. CentOS节点配置

在所有节点上编辑/etc/hosts文件

vim /etc/hosts

添加以下内容：

192.168.88.101 node1
192.168.88.102 node2
192.168.88.103 node3

重要： 确保所有节点上的/etc/hosts文件内容一致

SSH免密登录配置

1. 生成SSH密钥

在每台机器上执行以下命令（以root用户）：

ssh-keygen -t rsa -b 4096

一路按回车使用默认设置

2. 配置互信

在每台机器上执行以下命令：

ssh-copy-id node1
ssh-copy-id node2
ssh-copy-id node3

需要输入各节点的root密码

3. 验证免密登录

测试从node1登录到node2：

ssh node2

应能直接登录而无需密码

Hadoop用户配置

1. 创建Hadoop用户

在每台机器上执行：

useradd hadoop
passwd hadoop  # 设置密码为123456

2. 配置Hadoop用户SSH互信

切换到hadoop用户：

su - hadoop

生成SSH密钥：

ssh-keygen -t rsa -b 4096

配置互信：

ssh-copy-id node1
ssh-copy-id node2
ssh-copy-id node3

需要输入各节点的hadoop用户密码

提示： 所有后续Hadoop操作都应使用hadoop用户进行，以确保安全

JDK环境部署

1. 创建安装目录

mkdir -p /export/server

2. 解压JDK安装包

tar -zxvf jdk-8u351-linux-x64.tar.gz -C /export/server

3. 创建软链接

ln -s /export/server/jdk1.8.0_351 /export/server/jdk

4. 配置环境变量

编辑/etc/profile文件：

vim /etc/profile

在文件末尾添加：

export JAVA_HOME=/export/server/jdk
export PATH=$PATH:$JAVA_HOME/bin

5. 使配置生效

source /etc/profile

6. 配置系统级Java软链接

rm -f /usr/bin/java
ln -s /export/server/jdk/bin/java /usr/bin/java

7. 验证安装

java -version
javac -version

应显示Java版本信息
