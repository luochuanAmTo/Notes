GitHub Pages 部署示例

免费 · 快速 · 无需服务器

将此代码推送到 GitHub 仓库，立即获得一个线上网站

部署步骤

  1. 创建 GitHub 仓库 (命名: 用户名.github.io)
  2. 复制此 HTML 代码保存为 index.html
  3. 将文件推送到 GitHub
  4. 开启 GitHub Pages 功能

终端操作示例：

# 初始化本地仓库 git init # 添加文件到仓库 git add index.html # 提交更改 git commit -m "Initial commit" # 关联远程仓库 (替换为你的仓库URL) git remote add origin https://github.com/用户名/用户名.github.io.git # 推送到 GitHub git push -u origin main

GitHub 设置步骤

  1. 进入你的 GitHub 仓库
  2. 点击 Settings → Pages
  3. 选择分支: main
  4. 选择文件夹: / (root)
  5. 点击 Save
√ 部署成功！ 你的网站已上线： https://用户名.github.io 访问链接: https://用户名.github.io
